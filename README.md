# Chinese Checkers (Web Rebuild)

A fully redesigned Chinese Checkers experience built for the web with Nuxt + Vue.

## Run locally

```bash
bun install
bun run dev
```

Open `http://localhost:3000`.

## Gameplay

- 2-player local game.
- Traditional star board generated from cube coordinates.
- Legal moves include adjacent steps and chained jumps.
- Win by moving all 10 marbles into the opposite home triangle.

## Stack

- Nuxt 4
- Vue 3 + TypeScript
- Tailwind CSS (base import)

## Scripts

- `bun run dev` – development server
- `bun run build` – production build
- `bun run test` – unit tests
